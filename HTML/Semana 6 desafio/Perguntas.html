Pesquisar a diferença entre o JavaScript e a linguagem JAVA.

Java e JavaScript são linguagens de programação bem diferentes. O Java é uma linguagem de propósito geral,
usada para criar diversos tipos de aplicações, desde softwares desktop até aplicativos Android. Já o JavaScript é uma linguagem de script interpretada, utilizada principalmente para adicionar interatividade a páginas web.

Quais empresas criaram o JavaScript e o JAVA, e quais são as que atualmente mantêm o desenvolvimento dessas linguagens?

JavaScript

JavaScript foi criado por Brendan Eich em 1995 enquanto ele trabalhava na Netscape Communications. A linguagem foi originalmente chamada de LiveScript, mas foi renomeada para JavaScript em 1996.

Atualmente, o desenvolvimento do JavaScript é liderado pela Ecma International, uma organização de padronização internacional. A Ecma International publica o padrão ECMAScript, que define a sintaxe e a semântica do JavaScript.

JAVA

O Java foi criado por uma equipe de engenheiros da Sun Microsystems, liderada por James Gosling, entre 1991 e 1995.

Após a aquisição da Sun Microsystems pela Oracle em 2010, a Oracle se tornou a responsável pela manutenção do Java. A Oracle publica o Java Development Kit (JDK), que é a implementação oficial da linguagem Java.
